# npm-build-version

## Install

```sh
yarn add npm-build-version --dev
```
or
```sh
npm i npm-build-version --save-dev
```
## Usage

Add script into your npm scripts:

```json
{
  "name": "your-project",
  "scripts": {
    "build-version": "npm-build-version"
  }
}
```

Run

```javascript
yarn build-version
```
